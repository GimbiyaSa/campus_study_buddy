üîß Setup Instructions

Required: Configure Google Client ID

To use this testing tool, you need to:

  1. Get a Google OAuth Client ID:
    • Go to Google Cloud Console
    • Create or select a project
    • Go to "APIs & Services" ‚Üí "Credentials"
    • Create "OAuth 2.0 Client ID" ‚Üí "Web application"
    • ‚ö†Ô∏è IMPORTANT - Add authorized JavaScript origins:
      • https://gimbiyasa.github.io (for GitHub Pages)
      • http://localhost:3000 (for local development)
      • http://localhost:5000 (for backend testing)
      • http://127.0.0.1:3000 (alternative localhost)
  2. Update this page:
    • Download this HTML file
    • Replace YOUR_GOOGLE_CLIENT_ID.apps.googleusercontent.com with your actual Client ID
    • Open the modified file in your browser
üö® Getting "origin_mismatch" error?
This means your Google OAuth Client ID doesn't allow requests from https://gimbiyasa.github.io.
Solution: Add https://gimbiyasa.github.io to your OAuth Client's "Authorized JavaScript origins" in Google Cloud Console.

üìñ How to Use This Token

üåê With curl

üìÆ With Postman
